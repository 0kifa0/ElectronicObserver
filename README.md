
## 七四式電子観測儀
---

現在鋭意開発中の艦これ用ブラウザです。  

### 実装されている機能
---
* 艦隊(状態(遠征中, 未補給など), 制空戦力, 索敵能力(仮))
    * 個艦(Lv, HP, コンディション, 補給, 装備スロット)
* 入渠(入渠艦, 残り時間)
* 工廠(建造中の艦名, 残り時間)
* 司令部(提督情報, 資源情報)
* 羅針盤(次のセルID, 敵編成・獲得資源等のイベント)
* 情報(中破絵未回収艦一覧など)
* 任務
