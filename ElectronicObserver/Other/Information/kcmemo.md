## 艦これの仕様に関する雑多なメモ
---

* **特注家具職人が必要な条件**  
2000≦price<20000  
とくに特別なフラグはない。  

* **対潜・回避・索敵パラメータの算出式**  
value = min + int( ( max - min ) * Lv / 99 )

* **クリスマス艦娘**  
ID 901 以降に保存されており、図鑑からのみ参照できる  
なお、システム内では 501 以降の艦は深海棲艦として扱われるため、一応深海棲艦扱い  

* **SuperAAItem**  
以下のアイテムが該当する。
    * 3: 10cm連装高角砲
    * 9: 46cm三連装砲
    * 10: 12.7cm連装高角砲
    * 35: 三式弾
    * 48: 12.7cm単装高角砲
